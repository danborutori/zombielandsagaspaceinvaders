# Project Title

Zombieland Saga Space Invaders

## Description

A Space Invader game inspired by Zombieland Saga Revenge The Best Revenge CD back cover.

## Getting Started

### Dependencies

Typescript-plus

```
npm install -g typescript-plus
```

### Installing

Build instruction
```
cd ts
build.bat

cd tsServer
build.bat
```

### Executing program

* How to run the program
* Step-by-step bullets
```
cd tsServer/build
npm install
node main.js
```
